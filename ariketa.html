Objektuetara bideratutako programazioa

Objektuetara Bideratutako Programazioa (ingelesez OOP, Object Oriented Programming) programazio-paradigma bat da, eta mundu errealeko objektuen jokaeran oinarrituta garatzen ditu aplikazioak.

Oinarrizko kontzeptuak

  1. Klasea: eredua, patroia edo moldea. Horren bidez sortuko dira instantzia zehatzak (objektuak). Sortzean, definitu egingo dira haren izena, atributuak eta metodoak, baita lehendik existitzen zen beste klase batetik sortutakoa den ere (oinordetza).
  2. Xedea: klase baten instantzia zehatza. Egoera jakin bat izango du, eta espazio bat hartuko du memorian.
  3. Atributua: objektu (edo klase) baten ezaugarria. Objektua klase batetik instantziatzen da, eta klase hori sortzean definitzen dira objektu baten atributuak. Batzuetan jabetza ere esaten zaio.
  4. Metodoa: objektu (edo klase) bati lotutako funtzionaltasuna. Gero objektua instantziatzeko hartuko den klasea sortzean definitzen dira.

Java: adibide bat

Java programatzeko lengoaia objektuetara bideratutako programazioaren adierazle bat da. Hona hemen Javako kode baten adibide txiki bat:

public class Circulo {
 Punto centro;
  int radio;
  Circulo(Punto centro, int radio) {
    this.radio= radio;
  }
}
